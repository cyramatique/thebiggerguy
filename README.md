# The Bigger Guy — Pitch Site

Pitch website for *The Bigger Guy*, a screendance music video directed by [Cyramatique](https://instagram.com/cyramatique) and [Juliana Rogala](https://instagram.com/juliannaartisticventures), with choreography by Becca Francis and original music by Rebekka Louise.

Live site: `https://[your-github-username].github.io/the-bigger-guy`

---

## Project

*The Bigger Guy* is a 4-minute screendance music video set inside a church interior, exploring themes of forgiveness, grace, and emotional expectation. The camera operates as a creative collaborator — finding in choreography and sacred space what neither could express alone.

**Collaborators**
- Directors: Elina (Cyramatique) & Juliana Rogala
- Choreographer & performer: Becca Francis
- Music: Rebekka Louise

## Deployment

This site is deployed via GitHub Pages from the `main` branch.

To update the site, commit changes to `main` — GitHub Pages will rebuild automatically within a minute or two.

**To deploy for the first time:**
1. Push this repository to GitHub
2. Go to Settings → Pages
3. Set source to `Deploy from a branch` → `main` → `/ (root)`
4. Save — the site will be live at `https://[your-username].github.io/[repo-name]`
